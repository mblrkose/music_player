# music_player
A simple music player with html js

<img width="1920" height="1008" alt="image" src="https://github.com/user-attachments/assets/68dd0079-459e-4478-9ac7-fdc19d91aa6c" />

You can add or remove the musics in side music.js file. 
In order to add a music file copy and paste codes in line 5 to 9. Change the information inside it. 
In order to remove a music just delete the code of that music. Each code of a music is 5 lines.
